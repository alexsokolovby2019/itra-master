Бизнес-процесс «Создать пользователя подсистемы АИТП»

Предназначен для создания пользователя системы. Используется для автоматического создания свойств пользователей системы. Процесс создает пользователя информационной базы, а также соответствующий ему элемент справочника «ПользователиАИТП».

В качестве параметров бизнес-процесса используются реквизиты, одноименные свойствам типа ПользовательИнформационнойБазы.
