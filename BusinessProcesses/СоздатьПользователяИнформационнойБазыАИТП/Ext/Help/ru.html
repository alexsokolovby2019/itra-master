Бизнес-процесс «Создать пользователя информационной базы»

Предназначен для создания пользователей информационной базы. Используется для автоматического создания пользователей системы.

В качестве параметров бизнес-процесса используются реквизиты, которые соответствуют одноименным свойствам типа ПользовательИнформационнойБазы.
