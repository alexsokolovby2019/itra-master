Бизнес-процесс «Создать пользователя АИТП»

Предназначен для создания элемента справочника «ПользователиАИТП». Используется для автоматического создания пользователей системы.

В качестве параметров бизнес-процесса используются нижеследующие реквизиты:

УникальныйИдентификатор - строковое представление уникального идентификатора пользователя информационной базы.

Наименование - понятное наименование пользователя системы.
