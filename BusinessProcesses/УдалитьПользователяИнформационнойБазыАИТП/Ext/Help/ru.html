Бизнес-процесс «Удалить пользователя информационной базы»

Предназначен для удаления пользователей информационной базы. Используется для автоматического удаления пользователей системы.

В качестве параметров бизнес-процесса используются нижеследующие реквизиты:

УникальныйИдентификатор - строковое представление идентификатора пользователя информационной базы.

Имя - имя пользователя информационной базы.

Для выполнения бизнес-процесса достаточно заполнить любой из перечисленных реквизитов.
