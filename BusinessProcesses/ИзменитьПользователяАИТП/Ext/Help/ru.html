Бизнес-процесс «Изменить пользователя АИТП»

Предназначен для изменения элемента справочника «ПользователиАИТП». Используется для автоматического изменения свойств пользователей системы.

В качестве параметров бизнес-процесса используются нижеследующие реквизиты:

УникальныйИдентификатор - строковое представление уникального идентификатора пользователя информационной базы.

Наименование - понятное наименование пользователя системы.

ПомеченДляУдаления - признак пометки на удаление.

Для изменения соответствующих реквизитов также необходимо установить значения реквизитов ИзменитьИмяРеквизита, например: ИзменитьНаименование, в Истина.
