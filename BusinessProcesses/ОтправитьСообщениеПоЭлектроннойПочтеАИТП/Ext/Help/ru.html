Бизнес-процесс «Отправить сообщение по электронной почте»

Предназначен для отправки сообщений без вложений по электронной почте. Кодировка сообщений - UTF8. Используется для уведомления ИТ-специалистов и пользователей системы.

В качестве параметров используются нижеследующие реквизиты:

ТипТекстаСообщения - может быть html или простой текст.

Тема - тема сообщения.

Тело - тело сообщения.

КоличествоПопыток - количество попыток отправки сообщения. При превышении количества неудачных попыток будет зарегистрирована ошибка.

ИнтервалМеждуПопытками - интервал в секундах, между повторными попытками отправки.

Получатели - табличная часть, содержит список контактов получателей сообщения.

ПолучателиКопии - табличная часть, содержит список контактов получателей копии сообщения.
