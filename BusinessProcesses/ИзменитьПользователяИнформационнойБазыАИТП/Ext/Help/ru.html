Бизнес-процесс «Изменить пользователя информационной базы»

Предназначен для изменения свойств пользователя информационной базы. Используется для автоматического изменения свойств пользователей системы.

В качестве параметров бизнес-процесса используются реквизиты, одноименные реквизитам типа ПользовательИнформационнойБазы. Для указания факта необходимости изменения определенного реквизита, необходимо установить значение соответствующего реквизита ИзменитьИмяРеквизита в Истина.
