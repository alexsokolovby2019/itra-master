Бизнес-процесс «Отправить HTTP-запрос»

Предназначен для отправки http-запросов. Используется для взаимодействия с другими системами. Поддерживаются GET и POST запросы. Тело запроса должно быть строкой. Используемая кодировка - UTF8.

В качестве параметров используются нижеследующие реквизиты:

НастройкиСоединения - настройки, включающие в себя учетные данные, параметры прокси-сервера и т.п..

URL - url web-сервера, которому отправляется запрос.

ТипЗапроса - http метод. Может принимать значения GET или POST.

ТелоОтвета - тело http ответа.

КоличествоПопыток - количество попыток отправки запроса. Если количество неудачных попыток превысит это число - будет сгенерирована ошибка.

ИнтервалМеждуПопытками - интервал в секундах, между повторными попытками отправки.

Параметры - табличная часть, содержит список параметров url для запроса.

Заголовки - табличная часть, содержит список заголовков запроса.
