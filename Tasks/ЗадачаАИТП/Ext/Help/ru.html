Задача «Задача ИТ-процесса»

Является объектом, содержащим информацию о том, что нужно сделать в точке маршрута бизнес-процесса, а также результат выполнения этих действий.

Реквизиты объекта:

ВидЗадачи - вид выполняемой задачи. Задача может быть следующих видов: действие - выполнение какого-либо действия, ОбработкаОшибки - действие по обработке ранее возникшей ошибки, ожидание - ожидание определенной даты и времени.

Описание - текстовое описание действия, которое необходимо выполнить.

ДатаВыполнения - дата и время окончания выполнения действий по задаче.

Результат - результат выполнения действий по задаче. Результат может быть положительным, отрицательным или ошибкой. Как правило устанавливается в процессе выполнения действий по задаче.

ДальнейшиеДействия - действия, которые необходимо предпринять после выполнения задачи. Реквизит ДальнейшиеДействия может принимать нижеследующие значения: продолжить, повторить, завершить, отменить. Данный реквзит как правило устанавливается в процессе выполнения действий по задаче и предназначен для управления маршрутом потока действий в бизнес-процессе.

Ошибка - содержит информацию об ошибке, возникшей в процессе выполнения действий по задаче.

ОшибкаДляОбработки - содержит информацию об ошибке, возникшей при выполнении действий по предыдущей задачи. Используется в задачах вида ОбработкаОшибки.

СпособВыполнения - способ выполнения задачи. Может быть интерактивным, если действия выполняются пользователем или автоматическим, если действия выполняются программно.

ОписаниеРезультата - описание результата выполнения действий по задаче. Как правило используется в интерактивных задачах.

Исполнитель - пользователь системы, который пометил задачу как выполненную.
